To access your services from outside the cluster, you will need to set up port-forwarding:

For the client application:
  kubectl port-forward -n {{ .Values.namespace }} service/{{ .Values.services.client.name }} 8080:{{ .Values.services.client.port }}
  Then visit: http://localhost:8080

For the API Gateway:
  kubectl port-forward -n {{ .Values.namespace }} service/{{ .Values.services.apiGateway.name }} 8081:{{ .Values.services.apiGateway.port }}
  Then visit: http://localhost:8081
